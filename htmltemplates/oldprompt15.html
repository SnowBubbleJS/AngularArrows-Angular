Angular JS

Take a look at the factory.js, we have the function myFactory with an array called ‘FriendsList’ with multiple objects. We want to use this ‘FriendsList’ and display just the names. Similar to how the controller was created, we first need to link our application to the factory using the 'getter' syntax. Then, we will add a .factory(‘myFactory’, myFactory) right below the .module, just like how you did it in controller.js. The first argument in the factory method is the name we want for our factory, and the second argument is a function that returns an object with data. For more information on how a factory should be implement, please refer to: https://github.com/johnpapa/angular-styleguide#factories.
