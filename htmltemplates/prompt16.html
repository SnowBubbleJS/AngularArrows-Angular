Now that we have our factory.js set up, we need our controller needs to receive the data from our factory so we can pass it along to our HTML file.

In the controller.js, we need to inject our factory so that our controller has access to the factories data. Right above where you declared your myController function, write the following code myController.$inject = [‘myFactory’] because the function we’re going to write is going to be dependent of the data from factory.js, we call it dependency injection.
