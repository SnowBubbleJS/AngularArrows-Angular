Data binding using 'this' part 1

In your controller.js let's not use $scope to bind our data, but this. Using this, will help avoid defining methods into your controller (they should go into factories, which will be discussed later). For now, take out the $scope as the first parameter in your MyController function. Instead write var vm = this; as the first line in your function. Because our controller is binded through this, which we set as vm, let's change all of our code to
vm.name = 'Bob'
vm.age = 30
vm.occupation = 'hacker'.
